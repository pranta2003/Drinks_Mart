## 🚀 Getting Started

These are generic steps — update them to match your project commands.

1. **Clone the repo**

```bash
git clone https://github.com/pranta2003/Drinks_Mart.git
cd Drinks_Mart
```

2. **Install dependencies** (if any)

```bash
# npm (if using Node)
npm install

# or yarn
# yarn install
```

3. **Run locally**

```bash
# npm
npm run dev

# or build + serve

npm run start
```
